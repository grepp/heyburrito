# Heyburrito

이 레포지토리는 [heyburrito](https://github.com/chralp/heyburrito)를 커스텀한 것입니다.

## Getting Started

1. heyburrito ec2 서버에 접속한다.
2. `docker-compose up -d` 명령으로 컨테이너를 띄운다
